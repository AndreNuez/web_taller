# web_taller
Proyecto web para Cascara Taller
--------------------------------

Página web diseñada para emprendimiento (taller de arte).
Consta de un index con una breve descripción, mapa con ubicación, enlaces a redes sociales.
También de dos secciones en donde se puede acceder a una breve descripción de los talleres y workshops ofrecidos, y un formulario de contacto.
La web está en su fase final de desarrollo, abierta a modificaciones estéticas y de contenido.
